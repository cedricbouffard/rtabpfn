# Find value column in data frame

Find value column in data frame

## Usage

``` r
find_value_column(df, date_col, item_id_col)
```

## Arguments

- df:

  Data frame to search

- date_col:

  Name of date column to exclude

- item_id_col:

  Name of item_id column to exclude

## Value

Name of the first numeric column or NULL

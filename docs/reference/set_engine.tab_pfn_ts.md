# Set the model engine for TabPFN Time Series

Set the model engine for TabPFN Time Series

## Usage

``` r
set_engine.tab_pfn_ts(x, eng = c("tabpfn_ts"), ...)
```

## Arguments

- x:

  A model specification

- eng:

  A single character string for the engine (always "tabpfn_ts")

- ...:

  Additional engine-specific arguments

## Value

An updated model specification

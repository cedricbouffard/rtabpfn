# Set the model engine for TabPFN

Set the model engine for TabPFN

## Usage

``` r
set_engine.tab_pfn(x, eng = c("tabpfn"), ...)
```

## Arguments

- x:

  A model specification

- eng:

  A single character string for the engine (always "tabpfn" for this
  model)

- ...:

  Additional engine-specific arguments

## Value

An updated model specification

# Create test data frame for forecasting horizon

Create test data frame for forecasting horizon

## Usage

``` r
create_test_dataframe(train_df, prediction_length, date_col, item_id_col)
```

## Arguments

- train_df:

  Training data frame

- prediction_length:

  Number of forecast steps

- date_col:

  Name of date column

- item_id_col:

  Name of item_id column

## Value

Data frame with forecast horizon

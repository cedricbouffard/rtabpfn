# Check if Unsupervised Extension is available

Checks if tabpfn-extensions unsupervised module is installed and
available.

## Usage

``` r
check_unsupervised_available()
```

## Value

Logical indicating if unsupervised extension is available

## Examples

``` r
if (FALSE) { # \dontrun{
check_unsupervised_available()
} # }
```

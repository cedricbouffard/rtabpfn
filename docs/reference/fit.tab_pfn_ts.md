# Fit a TabPFN Time Series model

Fit a TabPFN Time Series model

## Usage

``` r
fit.tab_pfn_ts(
  x,
  formula = NULL,
  data = NULL,
  control = parsnip::control_fit(),
  ...
)
```

## Arguments

- x:

  A model specification

- formula:

  A formula specifying the model

- data:

  A data frame

- control:

  A \`parsnip::control_fit()\` object

- ...:

  Additional arguments

## Value

A fitted model object

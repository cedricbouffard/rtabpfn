# Create TimeSeriesDataFrame from R data frame

Create TimeSeriesDataFrame from R data frame

## Usage

``` r
create_timeseries_dataframe(df, date_col, value_col, item_id_col)
```

## Arguments

- df:

  R data frame

- date_col:

  Name of date column

- value_col:

  Name of value column

- item_id_col:

  Name of item_id column

## Value

Python TimeSeriesDataFrame object

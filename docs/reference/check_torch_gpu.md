# Check PyTorch GPU Status

Check if PyTorch is using GPU

## Usage

``` r
check_torch_gpu()
```

## Value

List with torch GPU status information (torch_available, cuda_available,
cuda_version, device_count, device_name)

## Examples

``` r
if (FALSE) { # \dontrun{
# Check if torch is using GPU
check_torch_gpu()
} # }
```
